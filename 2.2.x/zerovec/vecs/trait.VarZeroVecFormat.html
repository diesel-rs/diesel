Trait VarZeroVecFormat

pub trait VarZeroVecFormat: 'static + Sized { }
Expand description

This trait allows switching between different possible internal representations of VarZeroVec.

Currently this crate supports three formats: Index8, Index16 and Index32, with Index16 being the default for all VarZeroVec types unless explicitly specified otherwise.

Do not implement this trait, its internals may be changed in the future, and all of its associated items are hidden from the docs.

Dyn Compatibility§

This trait is not dyn compatible.

In older versions of Rust, dyn compatibility was called "object safety", so this trait is not object safe.
