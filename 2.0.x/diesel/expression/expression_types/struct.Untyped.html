Struct diesel::expression::expression_types::Untyped

source · 
pub struct Untyped;
Expand description

Query nodes with this expression type do not have a statically at compile time known expression type.

An example for such a query node in diesel itself, is sql_query as we do not know which fields are returned from such a query at compile time.

For loading values from queries returning a type of this expression, consider using #[derive(QueryableByName)] on the corresponding result type.
