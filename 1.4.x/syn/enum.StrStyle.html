pub enum StrStyle {
    Cooked,
    Raw(usize),
}
Expand description

The style of a string literal, either plain quoted or a raw string like r##"data"##.

Variants

Cooked

An ordinary string like "data".

Raw(usize)

A raw string like r##"data"##.

The unsigned integer is the number of # symbols used.
