Macro variant

Expand description

A macro allowing for compile-time construction of valid Variant subtags.

§Examples

Parsing errors don’t have to be handled at runtime:

assert_eq!(
  icu_locale_core::subtags::variant!("posix"),
  "posix".parse::<icu_locale_core::subtags::Variant>().unwrap()
);

Invalid input is a compile failure:

ⓘ
icu_locale_core::subtags::variant!("yes");
