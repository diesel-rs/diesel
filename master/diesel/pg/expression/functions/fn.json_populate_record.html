Available on crate feature postgres_backend only.
Expand description

This function json_populate_record takes a Record base and Json as an input and converts it to top-level JSON object to a row having the composite type of the base argument.

ยงExample


let expected: Value = serde_json::json!({
    "f1": "Alice",
    "f2": 16
});
let record: (String, i32) = diesel::select(json_populate_record::<Record<(Text, Integer)>, Json, _, _>(
        sql::<Record<(Text, Integer)>>("ROW('John', 30)"),
        expected
)).get_result(connection)?;
assert_eq!(record, ("Alice".to_string(), 16));
