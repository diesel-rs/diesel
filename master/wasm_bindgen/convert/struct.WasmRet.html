Struct WasmRet 

Source 
#[repr(C)]
pub struct WasmRet<T: WasmAbi> { /* private fields */ }
Expand description

A repr(C) struct containing all of the primitives of a WasmAbi type, in order.

This is used as the return type of imported/exported functions. WasmAbi types aren’t guaranteed to be FFI-safe, so we can’t return them directly: instead we return this.

If all but one of the primitives is (), this corresponds to returning the remaining primitive directly, otherwise a return pointer is used.

§⚠️ Unstable

This is part of the internal convert module, no stability guarantees are provided. Use at your own risk. See its documentation for more details.
