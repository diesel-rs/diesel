Struct IdentString 

Source 
pub struct IdentString { /* private fields */ }
Expand description

A wrapper for an Ident which also keeps the value as a string.

This struct can be used to perform string comparisons and operations.

With the optional serde feature, this will be serialized as a string and supports being deserialized from a string.
